INFRAMER

Every DevOps team manages infrastructure distributed across different databases - chef, icinga, vmware, etc. The information in each database is queryable through their APIs but there isn't one roof under which you can look at all the information about a node.Neither can you do a diff of different databases and answer questions like - How may cheffed nodes are not being monitored through icinga? Inframer is an open source python based tool which we built at BlueJeans Network to address these issues.

GOBLIN

Goblin is an automated resiliency testing tool that is used for determining the resiliency of a cloud based system. It allows generation of a load on the system, has hooks to cause various types of failures and methods to validate the system behaviour, and finally recovers the system back to normal. It has built-in support for testing resiliency of systems that use Zookeeper, Couchbase, Cassandra etc, and can be extended to other components as well.
